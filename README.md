# Smart-Data-Handling-Customer-Data-Pre-processing-and-Visualization

📊 Customer Data Preprocessing & Analysis
📌 Overview

This project focuses on transforming raw customer data into structured, meaningful insights through data preprocessing, feature engineering, and visualization. It demonstrates the complete pipeline from handling messy CSV data to generating actionable analytics.

⚙️ Key Features
🔹 Data Preprocessing

Converted raw customer data from CSV to JSON format
Designed nested data structures for better organization
Handled inconsistencies such as missing and invalid values
Ensured clean and structured datasets for further analysis

🔹 Data Segmentation
Split processed data into separate JSON files:

🧓 Retired customers
👨‍💼 Employed customers
⚠️ Customers with problematic credit card data

🔹 Feature Engineering

Created a new metric: Salary-Commute Ratio
Ranked and sorted customers based on this metric
Enabled deeper insights into income vs commuting efficiency

🔹 Data Visualization
Used Pandas, Seaborn, and Matplotlib for analysis
Explored relationships between:

Salary 💰
Age 🎂
Commute distance 🚗

Generated both univariate and multivariate visualizations

🛠️ Tech Stack

Programming: Python
Libraries:
Pandas
Seaborn
Matplotlib

Data Formats: CSV, JSON

📈 Insights Generated

Distribution of customer age and salary
Relationship between commute distance and income
Identification of data anomalies and inconsistencies
Improved understanding of customer segmentation
